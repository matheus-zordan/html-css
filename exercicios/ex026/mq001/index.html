Aprendendo media query

Para otimizar o estudo, foram criados dois arquivos CSS distintos, um projetado para carregar em dispositivos com tela e o outro destinado ao modo de impressão. Isso permite que o conteúdo seja apresentado de forma ideal, adaptando-se automaticamente ao tipo de dispositivo em uso. Para ver a aplicação do conceito funcionando, experimente tentar imprimir a página.
